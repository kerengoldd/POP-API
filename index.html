Users - User Sign Up

/user

Verb: POST

Example Request:
        {
        "email":"keren@gmail.com",
        "password":"*******",
        "firstName":"keren",
        "lastName":nina"
        "birthdate":"31/12/1990"
        }
Parameter
Field Type Description
email String User email to authentication
password String User password must contain 6 characters
firstName String User first name
lastName String User last name
birthdate Date User date of birth
